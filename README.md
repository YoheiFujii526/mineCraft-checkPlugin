# mineCraft-checkPlugin（Item Checklist）

自動仕分け機を作っている方向けに、「仕分け可能（チェック済み）なアイテム」を一覧で管理・確認できる Paper プラグインです。  
管理者がチェックリスト上でアイテムをチェックすると、プレイヤーの所持アイテムにチェック状態が表示されます。

---

## ✅ 対応環境

- **Paper**: 1.21.11（他の1.21系でも動作する可能性あり）
- **Java**: 21

---

## ✨ 機能

- 全アイテムのチェックリストGUIを表示
- アイテムをクリックしてチェック状態を切り替え（チェック済み/未チェック）
- ページ送り（Prev / Next）
- チェック済み数の表示
- チェック済みアイテムは、所持アイテムの表示に `[@:✓]` が付きます  
  ※アイテム名の言語はプレイヤー側の言語設定に依存します

---

## 🛠 導入方法

1. Releases から `.jar` をダウンロード
2. サーバーの `plugins/` に `.jar` を入れる
3. サーバーを再起動する

---

## 📖 使い方

### コマンド
- `/checklist` : チェックリストGUIを開く

### GUI操作
- アイテムをクリック：チェック状態を切り替え
- `Prev` / `Next`：ページ切り替え

---

## 💾 データ保存

チェック状態はサーバー側に保存されます。

- 保存先：`plugins/<プラグイン名>/sorted.yml`

---

## ⚠ 注意

- アイテム名の日本語表示は、プレイヤー側の言語設定（日本語/英語など）に依存します。
- 本プラグインは実験的な実装を含むため、問題があれば Issue で報告してください。
